# Yılanlı-Aybakan Hukuki Danışmanlık — Web Sitesi

İstanbul Bakırköy'de faaliyet gösteren avukatlık bürosunun tanıtım sitesi.
Tamamen statik HTML/CSS/JS — framework ve backend yoktur.

## Sayfalar

| Dosya | İçerik |
|---|---|
| `index.html` | Anasayfa — hero, avukatlar, hizmet özeti, SSS |
| `about.html` | Hakkımızda — büro tanıtımı, avukat profilleri |
| `services.html` | Hizmet alanları (detaylı) |
| `arabuluculuk.html` | Arabuluculuk hizmeti |
| `makaleler.html` | Hukuki makaleler (accordion + filtre) |
| `contact.html` | İletişim |
| `bakirkoy-avukat.html` vb. | Bölge SEO sayfaları (6 ilçe) |

## Geliştirme

Herhangi bir build adımı yoktur; `index.html` dosyasını tarayıcıda açmak yeterlidir.
Tasarım `style.css`, etkileşimler `script.js` içindedir.

Ayrıntılı proje notları için `.claude/SITE_OZET.md` dosyasına bakın.
